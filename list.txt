Чек-лист "Личные события"

1. Отображение личных событий в расписании => Личные события отображаются

2. Создание личного события
	- Переход к личным событиям
		1.Нажатие “+” для добавления => Открывается попап “Создание события”
		2.Нажатие пустой ячейки в расписании для добавления => Открывается попап “Создание события”
		3. Нажатие кнопки “Личное событие” в попап => Переходит в попап создания личного события
		4. Закрытие попап “Создание события” нажатием крестика => Попап закрывается
		5. Закрытие попап создания личного события нажатием крестика => Попап закрывается
	- Поле “Название события”
		1. Кириллица (Выставка) => Значение принимается
		2. Латиница (Conference) => Значение принимается
		3. С цифрами (Собрание13) => Значение принимается
		4. С пробелами (Важное событие) => Значение принимается
		5. От 1 до 40 символов (О, Электроаккустика, Lorem ipsum dolor sit amet, consectetuer) => Значение принимается
		6. Знаки препинания (Важно! Срочно посмотреть.) => Значение принимается
		7. Пустое поле => Значение не принимается
		8. Только пробелы => Значение не принимается
		9. Более 40 символов => Значение не принимается	
	- Поле “День” и “Время” (выбор даты при создании события через “+”)
		1. Личное событие за прошлый год, любое время => Дата выбирается
		2. Личное событие за прошлый месяц, любое время => Дата выбирается
		3. Личное событие вчера, любое время => Дата выбирается
		4. Личное событие сегодня, но время уже прошло => Дата выбирается
		5. Личное событие сегодня, прямо сейчас => Дата выбирается
		6. Личное событие сегодня, время еще не подошло => Дата выбирается
		7. Личное событие завтра, любое время => Дата выбирается
		8. Личное событие в следующем месяце, любое время => Дата выбирается
		9. Личное событие в следующем году, любое время => Дата выбирается
		10. Личное событие с некорректной датой, время любое => Дата не выбирается
		11. Личное событие с неустановленной датой, время любое => Дата не выбирается
	- Поле “День” и “Время” (выбор даты кликом по рассписанию)
		1. Личное событие за прошлый год, любое время => Дата выбирается
		2. Личное событие за прошлый месяц, любое время => Дата выбирается
		3. Личное событие вчера, любое время => Дата выбирается
		4. Личное событие сегодня, но время уже прошло => Дата выбирается
		5. Личное событие сегодя, прямо сейчас => Дата выбирается
		6. Личное событие сегодня, время еще не подошло => Дата выбирается
		7. Личное событие завтра, любое время => Дата выбирается
		8. Личное событие в следующем месяце, любое время => Дата выбирается
		9. Личное событие в следующем году, любое время => Дата выбирается
		10. Личное событие с некорректной датой, время любое => Дата не выбирается
		11. Личное событие с неустановленной датой, время любое => Дата не выбирается
	- Продолжительность личного события
		1. Указать интервал 10 мин => Интервал принимается
		2. Указать интервал 12 часов => Интервал принимается
		3. Указать интервал 23 часов 50 минут => Интервал принимается
		4. Ввод любых значений с клавиатуры => Данные вводятся
		5. Указать интервал 0 минут => Интервал не принимается
	- Поле “Описание”
		1. Кириллица (Это очень важное событие для моего ученика) => Значение принимается
		2. Латиница (This is a very important event for my student) => Значение принимается
		3. С цифрами (Это 24 съезд партии тестировщиков) => Значение принимается
		4. Со спецсимволами (®Собрание) => Значение принимается
		5. С ссылкой (https://skyengpublic.notion.site/282e0bc0c4c342bd8081a8c34fcec026) => Значение принимается
		6. С маркдауном (+ Проводник) => Значение принимается
		7. С изображением => Значение не принимается
	- Выбор цвета события
		1. Проверка цвета по умолчанию => Цвет серый
		2. Выбрать зеленый цвет => Цвет выбирается
	- Сохранение
		1. Заполнить все обязательные поля и нажать сохранить => Событие сохраняется

3. Действия над созданным личным событием
	- Редактирование
		1. Возможность редактирования поля Название события => Изменения принимаются
		2. Возможность редактирования поля День => Изменения принимаются
		3. Возможность редактирования поля Время => Изменения принимаются
		4. Возможность редактирования поля Описание => Изменения принимаются
		5. Возможность редактирования цвета события => Изменения принимаются
	- Перенос даты личного события
		1. Перенести личное событие из будущего в прошлое => Изменения принимаются
		2. Перенести личное событие из прошлого в будущее => Изменения принимаются
		3. Перенести личное событие из настоящего в прошлое => Изменения принимаются
		4. Перенести личное событие из настоящего в будущее => Изменения принимаются
	- Удаление
		1. Удаление существующего личного события из прошлого => Событие удаляется
		2. Удаление существующего личного события из будущего => Событие удаляется
		3. Удаление существующего личного события из настоящего => Событие удаляется

4. Взаимодействие личных событий с другими событиями расписания
	- Взаимодействие с уроками
		1. Урок и личное событие в одно время => Урок отображается всегда выше всего
	- Взаимодействие с другими личными событиями
		1. Личные события созданы на одно и тоже время => Отображается выше то, которое было создано последним
